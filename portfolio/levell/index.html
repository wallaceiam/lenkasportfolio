Levell

A Mobile App enabling users to prevent burnout, increase performance and improve their overall well-being

Project Overview

The Brief

  • Create a brand new mobile native app
  • Incorporate user experience (UX) in order to enhance user satisfaction with the product
  • Improve product usability, accessibility and pleasure
  • Define and design core functions and the overall product architecture
  • Engage their target audience whilst interacting with their product
  • Conduct analysis of the existing Levell website and understanding how people currently use their product
  • Increase the product value
  • Encourage the users to visit the app daily and input data on regular basis

The Problem

  • Only a desktop version is available for the users to login into Levell website to input data
  • Users get an email prompt rather than mobile notification to use Levell platform
  • The content of their website is mainly focused on business development rather than on individual user
  • The user can only input data against four daily drivers (mood, stress, energy and motivation), capture blockers and view simple trends in a dashboard
  • No extra features added that promote interaction with the user
  • Lack of value for the user to return on daily basis
  • Differentiate from other competitors on the market
  • Not enough data break-down for the user to understand the patterns of their feelings/behaviours

The Solution

What was initially a desktop app to input data was changed into a native mobile app that can be used on the go with additional features to promote habit/routine building, learning new skills, personalised support, and have a better overview of their well-being trends and easier capture of data.

Without a doubt, this app now brings more value to their users and is more likely to increase regular use and downloads. I believe using this app can create a positive future for individuals.
